the best times-table review app for Windows

What is "speedtable"?
speedtable is a Python-based Windows app, made for kids to be able to practice their times-tables (multiplication review) on their computer. It comes pre-compiled as "speedtable.exe", ready for use out-of-the-box (or really, out-of-the-downloads folder).
WARNING: The precompiled version ("speedtable.exe") ONLY works on Windows OSes (operating systems). If you're on OSX or Linux, see the compilation instructions.
